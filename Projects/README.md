# Анализ внутриигровых покупок в игре «Секреты Темнолесья»

## Описание проекта
Проект выполнен в рамках курса по SQL. Цель работы — изучить влияние характеристик игроков и их персонажей на покупку внутриигровой валюты «райские лепестки», а также оценить активность игроков при совершении внутриигровых покупок.

**Задачи исследования:**
- Провести разведочный анализ данных об игроках и покупках
- Рассчитать долю платящих игроков и её зависимость от расы персонажа
- Изучить статистику внутриигровых покупок (количество, сумма, распределение)
- Выявить популярные эпические предметы
- Оценить зависимость активности покупок от расы персонажа
- Сформулировать рекомендации для маркетинговой команды

## Навыки и инструменты
- **SQL** (PostgreSQL) — написание аналитических запросов
- **DBeaver** — подключение к базе данных, выполнение запросов
- **Статистика** — расчёт средних, медиан, стандартного отклонения, долей
- **Метрики:** доля платящих игроков, частота покупок, средний чек, LTV по расам

## Структура базы данных
Проект выполнен на базе данных `data-analyst-fantasy`, схема `fantasy`. База содержит 7 таблиц:

| Таблица | Описание |
|---------|----------|
| `users` | Информация об игроках (id, ник, класс, раса, страна, признак платящего) |
| `events` | Данные о покупках (id покупки, id игрока, дата, время, предмет, сумма) |
| `classes` | Справочник классов персонажей (13 классов) |
| `race` | Справочник рас персонажей (7 рас) |
| `country` | Справочник стран игроков |
| `skills` | Справочник легендарных умений (181 умение) |
| `items` | Справочник эпических предметов (184 предмета) |

**Связи между таблицами:**
- `users.class_id` → `classes.class_id`
- `users.race_id` → `race.race_id`
- `users.loc_id` → `country.loc_id`
- `users.ch_id` → `skills.ch_id`
- `events.item_code` → `items.item_code`

## Итоговые выводы и рекомендации

### Основные результаты:
1. **Доля платящих игроков** составляет **18%** от всех зарегистрированных пользователей. Наибольшая концентрация платящих игроков — в расе Human (абсолютные значения), наибольшая доля — у рас Angel и Demon.
2. **Активность покупок** слабо зависит от расы — доля покупателей и платящих стабильна по всем расам.
3. **Топ-предметы** — Book of Legends и Bag of Holding — формируют более 90% всех покупок и используются более чем 80% игроков.
4. **Раса Northman** показывает наилучшую монетизацию: самую высокую среднюю стоимость покупки и вторую по величине среднюю сумму на игрока.

### Рекомендации для маркетинговой команды:

1. **Для рас с высоким потенциалом монетизации (Demon, Angel):**
   - Провести ивенты, направленные на конвертацию неплатящих игроков в платящих
   - Выпустить эксклюзивные косметические предметы для этих рас для тестирования спроса

2. **Для расы Northman:**
   - Изучить причины высокой платёжеспособности игроков этой расы
   - Проанализировать, какие предметы покупают игроки Northman чаще всего
   - Рассмотреть возможность создания премиум-контента, ориентированного на эту расу

3. **По предметам:**
   - Изучить механику предметов Book of Legends и Bag of Holding — почему они так популярны
   - На их основе разработать новые предметы с похожими механиками
   - Внедрить систему временных предметов для тестирования новых идей без перегрузки основного ассортимента

4. **По менее востребованным предметам:**
   - Провести анализ актуальности — возможно, некоторые предметы требуют балансировки
   - Рассмотреть возможность улучшения их характеристик или замены на более востребованные

**Важно:** Все изменения вводить постепенно, с мониторингом обратной связи от игроков и отслеживанием ключевых метрик, чтобы не нарушить игровой баланс.
