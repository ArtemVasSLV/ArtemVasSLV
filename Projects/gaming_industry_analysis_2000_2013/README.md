# Анализ игровой индустрии (2000-2013)

## Описание проекта
Проект выполнен для команды игры «Секреты Темнолесья». Коллеги готовят статью о развитии игровой индустрии в начале XXI века для известного IT-ресурса. Цель статьи — привлечь внимание аудитории, интересующейся ретро-играми, и заинтересовать их проектом «Секреты Темнолесья».

**Задачи исследования:**
- Провести предобработку и подготовить данные к анализу
- Отобрать данные за период 2000-2013 годы
- Категоризовать игры по оценкам пользователей и критиков
- Выделить топ-7 платформ по количеству выпущенных игр

**Структура данных:**
- `Name` — название игры
- `Platform` — название платформы
- `Year of Release` — год выпуска игры
- `Genre` — жанр игры
- `NA sales`, `EU sales`, `JP sales`, `Other sales` — продажи по регионам (млн копий)
- `Critic Score` — оценка критиков (0-100)
- `User Score` — оценка пользователей (0-10)
- `Rating` — рейтинг ESRB

## Используемые библиотеки
- **pandas**, **numpy** — обработка и анализ данных

## Основные этапы работы

### 1. Предобработка данных
- Приведены названия столбцов к стилю snake_case
- Преобразованы типы данных в числовых столбцах
- Обработаны пропуски: удалены строки с пропусками в `name`, `genre`, `eu_sales`, `jp_sales`, `year_of_release`
- Обнаружено и удалено 179 дубликатов (1.07% от данных)
- Итоговый размер данных: 16 669 строк

### 2. Фильтрация данных
- Отобраны игры за период **2000-2013 годы**
- В выборку попало **12 971 игра** (78% от обработанных данных)
- Пик выпуска игр пришёлся на 2008-2009 годы

### 3. Категоризация по оценкам
**Оценки пользователей:**
- Высокая: 8-10 баллов — **2 303 игры** (17.8%)
- Средняя: 3-8 баллов — **4 145 игр** (31.9%)
- Низкая: 0-3 балла — **117 игр** (0.9%)
- Нет оценки — **6 406 игр** (49.4%)

**Оценки критиков:**
- Высокая: 80-100 баллов — **1 706 игр** (13.2%)
- Средняя: 30-80 баллов — **5 499 игр** (42.4%)
- Низкая: 0-30 баллов — **55 игр** (0.4%)
- Нет оценки — **5 711 игр** (44.0%)

### 4. Топ-7 платформ по количеству игр (2000-2013)

## Итоговые выводы
1. После предобработки получен качественный срез данных за 2000-2013 годы (12 971 игра)
2. Около половины игр не имеют пользовательских и критических оценок (вероятно, менее популярные игры)
3. Среди оценённых игр преобладают средние оценки (31.9% у пользователей, 42.4% у критиков)
4. Топ-7 платформ периода: PS2, DS, Wii, PSP, X360, PS3, GBA
5. PS2 и DS с большим отрывом лидируют по количеству игр (>2 100 игр каждая)
6. Портативные консоли (DS, PSP, GBA) составляют почти треть от всех игр в топ-7
